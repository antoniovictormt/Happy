## :green_book: How to run

### Clone Repository
$ git clone https://github.com/antoniovictormt/happy.git

### Run Web Project

* Install Dependencies
<br>$ yarn install

* Run Aplication
<br>$ yarn start

Go to http://localhost:3000/ to see the result.
